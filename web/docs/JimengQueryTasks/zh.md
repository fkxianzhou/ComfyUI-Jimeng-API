# 即梦视频生成任务列表查询

查询符合条件的即梦 API 视频生成任务列表

## 输入 (Inputs)

- **client (客户端)**: 即梦 API 客户端
- **page_num (页码)**: 查询的页码
- **page_size (每页数量)**: 每页返回的任务数量
- **status (任务状态)**: 要过滤的任务状态
  - `全部状态`
  - `成功`
  - `失败`
  - `运行中`
  - `排队中`
  - `已取消`
- **task_ids (任务 ID 列表)**: 要查询的特定任务 ID（每行一个）
- **model_choice (模型选择)**: 按特定模型过滤
- **custom_model_id (自定义推理接入点 ID)**: 按自定义模型 ID 过滤（优先于模型选择）

## 输出 (Outputs)

- **查询结果**: 查询到的任务列表（JSON 字符串）
- **任务数量**: 符合条件的总任务数量
