# 即梦参考图转视频

即梦 AI 参考图转视频节点。根据输入的提示词和一张或多张参考图像来生成视频。

## 输入 (Inputs)

- **client (客户端)**: 即梦 API 客户端
- **prompt (提示词)**: 用于视频生成的文本提示词
- **duration (时长 (秒))**: V 视频的期望时长（秒）
- **resolution (分辨率)**: 视频的分辨率（480p, 720p）
- **aspect_ratio (宽高比)**: 视频的宽高比
- **seed (种子)**: 随机种子，-1 表示随机
- **non_blocking (非阻塞模式)**: 启用后，任务将提交到后台执行，工作流将暂停。再次运行时会检查任务状态。
- **ref_image_1 (参考图 1)**: 参考图像 1（至少需要一张）
- **ref_image_2 (参考图 2)**: 可选的参考图像 2
- **ref_image_3 (参考图 3)**: 可选的参考图像 3
- **ref_image_4 (参考图 4)**: 可选的参考图像 4

## 输出 (Outputs)

- **视频**: 生成的视频文件
- **最后一帧**: 生成的视频的最后一帧图像
- **响应**: API 的 JSON 格式响应
