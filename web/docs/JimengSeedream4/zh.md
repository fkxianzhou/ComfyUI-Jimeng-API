# 即梦 Seedream 4

使用即梦 Seedream 4 模型生成图像，支持单图或组图（自动）模式。

## 输入 (Inputs)

- **client (客户端)**: 即梦 API 客户端
- **prompt (提示词)**: 用于图像生成的文本提示词
- **generation_mode (生成模式)**: 选择单图或图组生成
  - `单图模式 (禁用组图)`
  - `组图模式 (自动)`
- **max_images (最大图片数)**: 在图组模式下生成的最大图像数量
- **size (尺寸)**: 预设的图像尺寸。如果选择 'Custom'，将使用 '宽度' 和 '高度'。
- **width (宽度)**: 自定义宽度（仅当尺寸为 'Custom' 时生效）
- **height (高度)**: 自定义高度（仅当尺寸为 'Custom' 时生效）
- **seed (种子)**: 随机种子，-1 表示随机
- **watermark (水印)**: 是否在生成的图像上添加水印
- **images (输入图像)**: 用于图生图的输入图像（最多 10 张）

## 输出 (Outputs)

- **图像**: 生成的图像（批处理）
- **响应**: API 的 JSON 格式响应
