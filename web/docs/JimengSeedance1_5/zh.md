# 视频生成（Seedance 1.5 Pro）

视频生成节点。支持纯文本（T2V）、图生视频（I2V）以及首尾帧转视频。Seedance 1.5 Pro 特别支持音频生成与 Draft 样片模式。

## 输入 (Inputs)

- **生成音频**: 开启后模型为视频生成同步音频。
- **智能时长**: 开启后模型自动控制时长，忽略手动设置。
- **时长 (秒)**: 视频的期望长度（4-12秒）。
- **启用 Draft 样片模式**: 启用后将生成低成本的 480p 预览视频，用于快速验证生成视频是否符合预期。此模式不支持离线推理功能。
- **样片任务 ID**: 填入样片任务 ID，以生成正式视频。填入此项时，提示词和输入图像将被忽略，但分辨率等参数生效。
- **文件名前缀**: 批量生成时保存文件的路径前缀。
- **非阻塞模式**: 启用后任务提交到云端异步执行。
- **离线推理模式**: 使用 'flex' 服务等级，成本更低但速度可能较慢。
- **首帧图像 / 尾帧图像**: 提供视频的起始和结束画面参考。

## 输出 (Outputs)

- **视频**: 生成的视频文件。
- **最后一帧**: 视频的最后一帧图像（注意：样片模式下该输出可能为空或通过本地提取获得）。
