# 即梦 Seedream 3

使用即梦 Seedream 3 模型进行文本到图像或图像到图像的生成。

## 输入 (Inputs)

- **client (客户端)**: 即梦 API 客户端
- **prompt (提示词)**: 用于图像生成的文本提示词
- **size (尺寸)**: 预设的图像尺寸。如果选择 'Custom'，将使用 '宽度' 和 '高度'。
- **width (宽度)**: 自定义宽度（仅当尺寸为 'Custom' 时生效）
- **height (高度)**: 自定义高度（仅当尺寸为 'Custom' 时生效）
- **seed (种子)**: 随机种子，-1 表示随机
- **guidance_scale (引导系数)**: 提示词的引导强度
- **watermark (水印)**: 是否在生成的图像上添加水印
- **image (输入图像)**: 用于图生图（Seededit 3）的可选输入图像

## 输出 (Outputs)

- **图像**: 生成的图像
- **响应**: API 的 JSON 格式响应
