# Jimeng Seedream 4 节点使用说明

该节点使用即梦最新的 Seedream 4.0/4.5 模型生成图像。支持灵活的单图或组图生成策略。

## 参数说明

### 核心参数
- **client**: 连接 `JimengAPIClient` 节点。
- **model_version**: 选择模型版本，推荐使用 `doubao-seedream-4.5`。
- **enable_group_generation**: **关键开关**。
    - **开启 (True)**: 进入组图模式，根据提示词和参考图生成一组连贯的图片。
    - **关闭 (False)**: 进入单图模式，生成单张高质量图片。
- **max_images**: 生成数量设置。仅在 `enable_group_generation` 开启时可见并生效。

### 规则详解

| 模式 | 开关状态 | 输入 | 输出数量 | 限制条件 |
| :--- | :--- | :--- | :--- | :--- |
| **文生图** | False | 仅提示词 | 1 张 | 无 |
| **图生图** | False | 提示词 + 参考图 (1-14张) | 1 张 | 无论输入几张参考图，均融合生成 1 张结果 |
| **文生组图** | True | 仅提示词 | N 张 | N ≤ 15 |
| **图生组图** | True | 提示词 + 参考图 (M张) | N 张 | M + N ≤ 15 |
