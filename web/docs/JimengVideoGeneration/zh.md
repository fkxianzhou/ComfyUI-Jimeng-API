# 即梦视频生成

即梦 AI 视频生成节点。支持纯文本生成（T2V），或使用首帧/尾帧图像进行视频生成（I2V）。

## 输入 (Inputs)

- **client (客户端)**: 即梦 API 客户端
- **model_choice (模型选择)**: 选择要使用的视频生成模型
- **prompt (提示词)**: 用于视频生成的文本提示词
- **duration (时长 (秒))**: 视频的期望时长（秒）
- **resolution (分辨率)**: 视频的分辨率（480p, 720p, 1080p）
- **aspect_ratio (宽高比)**: 视频的宽高比
- **camerafixed (固定镜头)**: 是否固定镜头
- **seed (种子)**: 随机种子，-1 表示随机
- **generation_count (生成数量)**: 一次性并发生成的任务数量。如果大于 1，将激活批量保存功能。
- **batch_save_path (批量保存路径)**: 当生成数量大于 1 时，所有视频和（可选的）尾帧将被保存到 ComfyUI 'output' 目录下的这个子路径中。
- **save_last_frame_batch (批量保存尾帧)**: 当生成数量大于 1 时，是否同时保存所有视频的最后一帧。
- **non_blocking (非阻塞模式)**: 启用后，任务将提交到后台执行，工作流将暂停。再次运行时会检查任务状态。
- **image (首帧图像)**: 用于图生视频的可选起始图像
- **last_frame_image (尾帧图像)**: 用于图生视频的可选结束图像（需要首帧）

## 输出 (Outputs)

- **视频**: 生成的视频文件（批量生成时为按种子排序的第一个）
- **最后一帧**: 生成的视频的最后一帧图像（批量生成时为按种子排序的第一个）
- **响应**: API 的 JSON 格式响应（批量生成时为所有任务的列表）
