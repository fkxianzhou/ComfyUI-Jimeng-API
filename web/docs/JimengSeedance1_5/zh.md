# 视频生成（Seedance 1.0 / 1.5 Pro）

视频生成节点。支持纯文本（T2V）、图生视频（I2V）以及首尾帧转视频。

## 输入 (Inputs)

- **模型版本**: 可选 1.0 Pro/Lite 或最新的 1.5 Pro 模型。
- **生成音频**: (仅 1.5 Pro) 为视频生成同步音频。
- **智能时长**: (仅 1.5 Pro) 开启后模型自动控制时长，忽略手动设置。
- **时长 (秒)**: 视频的期望长度（4-12秒）。
- **文件名前缀**: 批量生成时保存文件的路径前缀。
- **非阻塞模式**: 启用后任务提交到云端异步执行。
- **离线推理模式**: 使用 'flex' 服务等级，成本更低但速度可能较慢。
- **首帧图像 / 尾帧图像**: 提供视频的起始和结束画面参考。

## 输出 (Outputs)

- **视频**: 生成的视频文件。
- **最后一帧**: 视频的最后一帧图像。
