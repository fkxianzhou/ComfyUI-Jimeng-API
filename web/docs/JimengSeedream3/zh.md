# 图像生成（Seedream 3）

使用 Seedream 3 模型进行文本到图像或图像到图像的生成。

## 输入 (Inputs)

- **客户端**: 火山方舟 API 客户端。
- **提示词**: 用于生成的文本描述。
- **尺寸**: 预设尺寸或选择 'Custom' 使用自定义宽高。
- **种子**: 随机种子，-1 表示随机。
- **引导系数**: 提示词的引导强度。
- **生成数量**: 一次生成的并发图像数。
- **输入图像**: (可选) 用于图生图（Seededit 3）的参考图。

## 输出 (Outputs)

- **图像**: 生成的结果。
- **响应**: API 的完整 JSON 响应。
